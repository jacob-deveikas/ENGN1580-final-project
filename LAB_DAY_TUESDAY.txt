ANTI-PANIC: WHAT IF NOTHING WORKS AT ALL?
================================================================

If the very first FSK 50 bps test (2a) fails:
1) Confirm system volume is 50-70%, not max, not muted.
2) Confirm Mic Mode = Standard.
3) Confirm both --device flags point to External in/out, not Built-in.
4) Confirm system Sound preferences show input/output as External.
5) Run python3 device_doctor.py
6) Run python3 install_check.py
7) Paste me everything.

If the dashboard window is blank or crashes:
1) pip install --upgrade pyqtgraph PyQt6
2) Try with --mode-audio loopback (no audio HW needed).
3) Paste me the error trace.

If two-laptop UDP backchannel never connects:
1) Disable both laptops' firewalls (System Settings → Network → Firewall).
2) Confirm both on same WiFi network (not eduroam isolation).
3) Try a USB-C ethernet cable between the two laptops as backup.
